# T007: テクニカル分析サービス実装

## 概要
各種テクニカル指標を計算するサービスクラスを実装する。

## 実行内容
- 移動平均線（SMA、EMA）計算機能を実装する
- MACD計算機能を実装する
- RSI計算機能を実装する
- ボリンジャーバンド計算機能を実装する
- テクニカル指標計算の共通インターフェースを作成する
- 計算エラーのハンドリングを実装する
